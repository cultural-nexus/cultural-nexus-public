# Privacy Policy

**Cultural Nexus Platform**
*Last updated: March 4, 2026*

## Information Collection

This application does not collect, store, or process any personal data from Pinterest users or any other third-party users.

## Application Purpose

Cultural Nexus is an internal, first-party automation tool used solely by its developers to publish curated exhibition content to owned-and-operated Pinterest accounts. It is not a consumer-facing application.

## Third-Party Authentication

This application does not allow external users to authenticate via Pinterest OAuth or any other third-party authentication mechanism. All API credentials are managed internally by the development team.

## Data Sharing

As no user data is collected, no user data is shared with or sold to third parties.

For questions about this privacy policy, contact: [privacy@cultural-nexus.com](mailto:privacy@cultural-nexus.com)
